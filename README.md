## 概要

**kurotan (くろたん)** は、Claude Code のセッション状況に連動してデスクトップに常駐するマスコットアプリです。
Claude Code が思考中なのか、ファイルを読んでいるのか、コマンドを実行しているのか、エラーで止まっているのか —
そういった "今この瞬間 Claude が何をしているか" を、デスクトップ右下にちょこんと現れた小さなマスコットの表情・モーション・吹き出しでリアルタイムに可視化します。

ターミナルを複数開いて Claude Code を並走させても、セッションごとに 1 匹のくろたんが並んで作業します。
どのセッションが何をしているかが一目で分かり、作業の相棒として、また場を和ませる生体ステータスインジケータとして机の片隅に住み着きます。

## 機能

- **マスコット状態の可視化** — Claude Code が今何をしているか (待機 / 考え中 / ツール実行 / 完了 / エラー / 終了 など) をマスコットの表情・モーションで示します
- **ツール別モーション** — `Read` / `Edit` / `Bash` / `WebFetch` / `Skill` 等、使っているツールに応じてポーズが変わります
- **子くろたん表示** — Claude Code が補助タスクを並列実行する時、親くろたんの横に小さな子くろたんが並んで一緒に作業します
- **マルチセッション並列表示** — 複数の Claude Code セッションを並走させると、セッションごとに 1 匹のくろたんが並びます
- **デスクトップ透過常駐** — マスコット以外の領域は背後アプリの操作を阻害しません
- **位置の永続化** — マスコットを好きな位置にドラッグすると、次回起動時も同じ位置に戻ります
- **設定画面** — アートスタイル切替、配置リセット、言語切替などをタスクトレイから呼び出せます
- **hooks 自動連携** — 初回起動時に Claude Code の hooks 設定 (`~/.claude/settings.json`) に自動マージ。設定画面から再インストール / アンインストールも可能
- **かんたんぼたん** — よく使うテキストをワンクリックで送信するフローティングバー (詳細は[下記](#かんたんぼたんの使い方))

## 起動後のくろたん

くろたんを起動すると **画面右下にくろたん本体** が出てきます。あわせて **タスクトレイ (画面右下の時計の左にある小さなアイコン群)** に kurotan のアイコンが常駐します。

ウィンドウのタイトルバーや「×」ボタンは無く、デスクトップに直接居着くスタイルです。

### タスクトレイのアイコンを右クリックすると

| 項目 | できること |
|---|---|
| **アクティブセッション: N 匹** | いま画面に出ているくろたんの数 (情報表示のみ) |
| **設定** | くろたんの動作 / 見た目 / 配色などを変える設定ウィンドウを開く |
| **全マスコットを閉じる** | 画面上のくろたんをすべて消す (再度 Claude Code を動かすと復活) |
| **すべて整列** | くろたんの位置を画面右下にきれいに整列し直す |
| **権限モード: Auto / Custom** | Claude Code がツールを使うときの確認方法を切り替え |
| **カスタム確認設定...** | Custom モードで「どのツールは確認・許可・拒否するか」を細かく決める |
| **About** | バージョン情報・非公式表記の確認 |
| **終了** | kurotan 自体を終了する (タスクトレイから消える) |

### くろたん本体を右クリックすると

| 項目 | できること |
|---|---|
| **セッション情報を表示** | そのくろたんが対応する作業フォルダ・モデル名を吹き出しで表示 |
| **このセッションを非表示** | このくろたんだけを消す (他のくろたんはそのまま) |
| **すべて整列** | 全くろたんを画面右下に整列し直す |
| **全マスコットを閉じる** | 画面上のくろたんをすべて消す |
| **終了** | kurotan 自体を終了 |

### くろたんを動かす

- **ドラッグで移動**: くろたんを左クリックしたままドラッグすると、自由な位置に動かせます。位置は再起動しても覚えています

<a id="smartscreen"></a>
### SmartScreen の表示について

インストーラを実行すると **「Windows によって PC が保護されました」** という青いダイアログが表示されます。これは未署名の配布物だからで、悪意のあるソフトではありません。

次の手順で進めてください:

1. ダイアログ左下の **「詳細情報」** をクリック
2. 下に表示される **「実行」** ボタンをクリック
3. インストールが始まります

### 既存の Claude Code セッションについて

くろたんを **後から起動した場合**、すでに動いている Claude Code セッションのうち、**直近 30 分以内に動きがあったもの** が画面に出ます。それより前から放置されているセッションは検出対象外です。

## かんたんぼたんの使い方

タスクトレイ → 「かんたんぼたんを表示」または、くろたんを右クリック → 「かんたんぼたんを表示」で起動します。

### 基本操作

| 操作 | 動作 |
|---|---|
| ＋ ボタン | 新しいボタンを追加 |
| ボタン右クリック | ラベル / 送信内容を編集 |
| 端をドラッグ | バーのサイズ変更 (形に応じてボタンが自動整列) |
| × ボタン | バーを閉じる |
| 日本語入力 | クリップボード経由で送信 (IME 確定対応) |

### 送信先

ボタン押下時、**その瞬間にフォアグラウンドにあるウィンドウ**へキー入力 + Enter が送信されます。VSCode に限らず、Windows Terminal / cmd / PowerShell / Notepad など任意のウィンドウで動作します。想定用途は **Claude Code CLI を動かしているターミナル**です。

### 🎯 フォーカス先 (ターゲットキャプチャ)

毎回フォーカスを合わせ直すのが面倒な時に使います:

1. かんたんぼたんの **🎯 ボタン** を押す
2. **3 秒カウントダウン** が始まるので、カウント中に送信したいウィンドウ (例: VSCode 内蔵ターミナル) の入力欄にカーソルを置く
3. カウント終了時のカーソル位置が **送信先として記憶** される
4. 以後のボタン送信時、自動で記憶座標へカーソル移動 + クリックして対象ウィンドウをフォーカスしてから送信

| 操作 | 動作 |
|---|---|
| 🎯 ボタン左クリック | 3 秒カウント後にカーソル位置を記録 |
| 🎯 ボタン右クリック | 記憶を解除 (フォアグラウンドへ直接送信に戻る) |

> **注**: フォーカス先を記憶している間は、別アプリを最前面にしていてもクリックで戻って送信されます。意図しないクリック動作を避けたい時は 🎯 右クリックで解除してください。

## ソースから起動 (開発者向け)

EXE インストーラを使わずソースから直接動かしたい場合の手順です。Node.js 18+ が必要です。

### 1. リポジトリを clone して依存をインストール

```bash
git clone https://github.com/fechirin-cyber/kurotan-app.git
cd kurotan-app
npm install
```

### 2. kurotan を起動

```bash
npm run dev
```

起動するとタスクトレイにアイコンが出て、デスクトップに最初のくろたんが控えめに登場します。

### 3. Claude Code 連携 hooks をインストール

```bash
node src/installer/install-hooks.js
```

ユーザーの `~/.claude/settings.json` に kurotan 連携の hooks 設定 (`SessionStart` / `PreToolUse` / `Stop` ほか計 10 イベント) が安全にマージ追記されます。
事前に変更内容を確認したい場合は `npm run install-hooks:dry`、後から取り外したい場合は `npm run uninstall-hooks` を使います。

### 4. 別ターミナルで Claude Code を起動

```bash
claude
```

セッションが立ち上がると、自動でデスクトップにくろたんが 1 匹出現し、Claude の作業に合わせてモーションが切り替わります。
複数ターミナルで `claude` を立ち上げれば、セッションごとに 1 匹ずつくろたんが増えていきます。

### 5. (任意) 自分でインストーラ EXE をビルド

```bash
npm run build
```

`dist/kurotan Setup <version>.exe` が生成されます。

## プライバシー / データの取扱い

- **外部送信なし**: 操作履歴・設定・生成物等のデータは利用者のローカル機器内にのみ保存され、開発者または第三者の運営する外部サーバーへ送信しません
- **ローカル通信のみ**: Claude Code との連携はローカルループバック (`127.0.0.1`) 上の内部通信のみ。インターネット経由の通信は一切行いません
- **テレメトリ / アナリティクスなし**: アクセス解析・利用統計・クラッシュレポート等の自動送信機能はありません
- **保存される情報**: 設定値 (アート / 言語 / 配置等) / 各セッションのマスコット位置 / ツール許可ダイアログの履歴 / hooks 経由で受信したセッション ID と作業ディレクトリ (識別子のみ、会話本文や認証情報は保持しません)
- **詳細条項**: [EULA.md 第 4 条](EULA.md) を参照

> 補足: Claude Code 本体は Anthropic のサーバーと通信します。これは Claude Code 側の挙動であり、本アプリの送信ではありません。Claude Code のプライバシーポリシーが適用されます。

## トラブルシューティング (FAQ)

### Q1. インストール時に「Windows によって PC が保護されました」と出ます

未署名配布のため発生する正常な警告です。「**詳細情報**」→「**実行**」で続行できます。詳細は [SmartScreen の表示について](#smartscreen) を参照してください。

### Q2. Claude Code を起動してもくろたんが出ません

以下を順に確認してください:

1. **タスクトレイに kurotan アイコンがあるか** — なければ kurotan 本体が未起動。スタートメニューから「kurotan」を起動
2. **Hooks がインストールされているか** — 設定画面 → Hooks Installer の状態を確認 (「✓ 登録済」となっているか)
3. **`~/.claude/settings.json` の整合性** — 他ツールと併用している場合、`kurotan-notify` の hook 行が残っているか確認
4. **ポート占有** — 別アプリが `127.0.0.1:47600` を使っていないか (kurotan は 47601-47610 にフォールバックします)

### Q3. くろたんが寝たまま起きません

くろたんの上にマウスカーソルを重ねると起きます。寝言が出る時はカーソルが当たっていない状態です。

### Q4. アプリのアイコンが古いままに見えます

Windows のアイコンキャッシュが古い情報を保持している場合があります。タスクマネージャーで `explorer.exe` を再起動するか、コマンドプロンプトで `ie4uinit.exe -show` を実行してください。

### Q5. 商用利用 / 配布したいです

本アプリは [EULA.md](EULA.md) で **非商用・個人利用限定** としています。販売・有償配布・広告組込・課金組込はお控えください。フォーク改変は MIT License の範囲内で歓迎しますが、その際はフォーク元としてご紹介ください。

### Q6. CPU / メモリ使用量が気になります

設定画面 → 「全マスコットを閉じる」で全マスコットを消すと、kurotan 本体は常駐したまま画面負荷を最小化できます。完全停止する場合はタスクトレイ → 「終了」。

### Q7. 言語を切り替えたいです

設定画面 → Language で **日本語 / English / 简体中文 / 繁體中文 / 한국어** の 5 言語に切り替えできます。
